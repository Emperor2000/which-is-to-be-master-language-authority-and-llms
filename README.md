## Overview

Programming languages operate according to formally specified semantics. Natural language does not. Its meaning depends on context, convention, perspective, and interpretation.

This document considers the architectural consequences of that distinction. It discusses prompt injection, information boundaries, authoritative state, context construction, and why instructions given to a language model should not be treated as enforceable security controls.
